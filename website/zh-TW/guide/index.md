# VMark 快速入門

VMark 是一款本地優先的 Markdown 編輯器，具備雙重編輯模式、豐富的排版工具，以及出色的中日韓文（CJK）支援。

## 快速開始

1. 從[下載頁面](/zh-TW/download)**下載並安裝** VMark
2. **啟動應用程式**，立即開始寫作
3. 使用 `Cmd/Ctrl + O` **開啟檔案**，或拖放 `.md` 檔案
4. 使用 `Cmd/Ctrl + Shift + O` **開啟資料夾** 以進入工作區模式

## 介面概覽

### 主要區域

- **編輯器**：撰寫文件的主要區域
- **側邊欄**：檔案樹狀導覽（使用 `Ctrl + Shift + 2` 切換顯示）
- **大綱**：文件結構檢視（使用 `Ctrl + Shift + 1` 切換顯示）
- **狀態列**：字數統計、字元計數與自動儲存狀態（使用 `F7` 切換顯示）
- **終端機**：整合的 shell 面板（使用 `` Ctrl + ` `` 切換顯示）

### 選單列

- **檔案**：新建、開啟、儲存、匯出操作
- **編輯**：還原/重做、剪貼簿、尋找/取代、文件歷史記錄
- **區塊**：標題、清單、引言、行操作
- **格式**：文字樣式、連結、文字轉換
- **檢視**：編輯器模式、側邊欄、專注/打字機模式
- **工具**：文字清理、中日韓文排版、圖片管理

### 編輯模式

VMark 支援兩種可切換的編輯模式：

| 模式 | 說明 | 快捷鍵 |
|------|------|--------|
| 富文字 | 所見即所得，即時格式預覽 | 預設 |
| 原始碼 | 帶有語法高亮的原始 Markdown | `F6` |

### 檢視模式

利用以下檢視模式提升寫作專注度：

| 模式 | 說明 | 快捷鍵 |
|------|------|--------|
| 專注 | 高亮顯示當前段落 | `F8` |
| 打字機 | 保持游標置中 | `F9` |
| 自動換行 | 切換行換行 | `Alt + Z` |

## 基本格式

### 文字樣式

| 樣式 | 語法 | 快捷鍵 |
|------|------|--------|
| **粗體** | `**文字**` | `Cmd/Ctrl + B` |
| *斜體* | `*文字*` | `Cmd/Ctrl + I` |
| ~~刪除線~~ | `~~文字~~` | `Cmd/Ctrl + Shift + X` |
| `程式碼` | `` `code` `` | `Cmd/Ctrl + Shift + `` ` `` |

### 區塊元素

- **標題**：使用 `#` 符號或 `Cmd/Ctrl + 1-6`
- **清單**：以 `-`、`*`、`1.` 或 `- [ ]` 開頭建立任務清單
- **引言**：以 `>` 開頭，或使用 `Alt/Option + Cmd + Q`
- **程式碼區塊**：使用三個反引號並可指定語言
- **表格**：使用「格式」選單或 `Cmd/Ctrl + Shift + T`

## 檔案操作

### 建立與開啟

- **新建檔案**：`Cmd/Ctrl + N`
- **開啟檔案**：`Cmd/Ctrl + O`
- **開啟資料夾**：`Cmd/Ctrl + Shift + O`（工作區模式）

### 儲存

- **儲存**：`Cmd/Ctrl + S`
- **另存新檔**：`Cmd/Ctrl + Shift + S`
- **自動儲存**：預設啟用，可在設定中調整

### 匯出

- **匯出 HTML**：使用 **檔案 → 匯出 HTML** — 包含互動式 VMark 閱讀器
- **匯出 PDF**：使用列印（`Cmd/Ctrl + P`）並另存為 PDF
- **複製為 HTML**：`Cmd/Ctrl + Shift + C`

匯出的 HTML 包含 VMark 閱讀器，具備目錄、設定面板等功能。[了解更多 →](/zh-TW/guide/export)

## 設定

使用 `Cmd/Ctrl + ,` 開啟設定進行自訂：

- **外觀**：主題、字型、字型大小、行距
- **編輯器**：自動儲存間隔、預設行為
- **檔案與圖片**：資產管理、文件工具
- **整合**：AI 供應商、MCP 伺服器
- **語言**：中日韓文排版規則
- **Markdown**：匯出選項、排版偏好設定
- **快捷鍵**：自訂鍵盤快捷鍵
- **終端機**：終端機字型大小與行距

## AI 寫作輔助

VMark 內建 AI 精靈 — 選取文字後按 `Mod + Y`，即可使用 AI 潤稿、擴寫、翻譯或改寫你的內容。在 **設定 > 整合** 中設定偏好的供應商。

[了解更多 AI 精靈 →](/zh-TW/guide/ai-genies) | [設定供應商 →](/zh-TW/guide/ai-providers)
