# VMark 시작하기

VMark는 이중 편집 모드, 풍부한 서식 도구, 탁월한 CJK (중국어/일본어/한국어) 지원을 갖춘 로컬 우선 마크다운 에디터입니다.

## 빠른 시작

1. [다운로드 페이지](/ko/download)에서 VMark를 **다운로드하여 설치** 합니다
2. **앱을 실행** 하고 바로 글쓰기를 시작합니다
3. `Cmd/Ctrl + O`로 **파일을 열거나** `.md` 파일을 드래그 앤 드롭합니다
4. 워크스페이스 모드를 위해 `Cmd/Ctrl + Shift + O`로 **폴더를 엽니다**

## 인터페이스 개요

### 주요 영역

- **에디터**: 문서를 작성하는 메인 영역
- **사이드바**: 파일 트리 탐색 (`Ctrl + Shift + 2`로 토글)
- **아웃라인**: 문서 구조 보기 (`Ctrl + Shift + 1`로 토글)
- **상태 표시줄**: 단어 수, 문자 수, 자동 저장 상태 (`F7`로 토글)
- **터미널**: 통합 셸 패널 (`` Ctrl + ` ``로 토글)

### 메뉴 바

- **파일**: 새 파일, 열기, 저장, 내보내기 작업
- **편집**: 실행 취소/다시 실행, 클립보드, 찾기/바꾸기, 문서 히스토리
- **블록**: 제목, 목록, 인용문, 줄 작업
- **서식**: 텍스트 스타일, 링크, 텍스트 변환
- **보기**: 에디터 모드, 사이드바, 포커스/타이프라이터 모드
- **도구**: 텍스트 정리, CJK 서식, 이미지 관리

### 편집 모드

VMark는 두 가지 편집 모드 간에 전환할 수 있습니다:

| 모드 | 설명 | 단축키 |
|------|------|--------|
| 리치 텍스트 | 실시간 서식이 적용되는 WYSIWYG 편집 | 기본값 |
| 소스 | 구문 강조가 있는 원시 마크다운 | `F6` |

### 보기 모드

다음 보기 모드로 글쓰기에 집중하세요:

| 모드 | 설명 | 단축키 |
|------|------|--------|
| 포커스 | 현재 단락 강조 | `F8` |
| 타이프라이터 | 커서를 중앙에 유지 | `F9` |
| 자동 줄 바꿈 | 줄 바꿈 토글 | `Alt + Z` |

## 기본 서식

### 텍스트 스타일

| 스타일 | 문법 | 단축키 |
|--------|------|--------|
| **굵게** | `**text**` | `Cmd/Ctrl + B` |
| *기울임꼴* | `*text*` | `Cmd/Ctrl + I` |
| ~~취소선~~ | `~~text~~` | `Cmd/Ctrl + Shift + X` |
| `코드` | `` `code` `` | `Cmd/Ctrl + Shift + `` ` `` |

### 블록 요소

- **제목**: `#` 기호 또는 `Cmd/Ctrl + 1-6` 사용
- **목록**: 줄을 `-`, `*`, `1.` 또는 작업 목록의 경우 `- [ ]`로 시작
- **인용문**: `>`로 시작하거나 `Alt/Option + Cmd + Q` 사용
- **코드 블록**: 선택적 언어와 함께 삼중 백틱 사용
- **테이블**: 서식 메뉴 또는 `Cmd/Ctrl + Shift + T` 사용

## 파일 작업

### 생성 및 열기

- **새 파일**: `Cmd/Ctrl + N`
- **파일 열기**: `Cmd/Ctrl + O`
- **폴더 열기**: `Cmd/Ctrl + Shift + O` (워크스페이스 모드)

### 저장

- **저장**: `Cmd/Ctrl + S`
- **다른 이름으로 저장**: `Cmd/Ctrl + Shift + S`
- **자동 저장**: 기본적으로 활성화, 설정에서 구성 가능

### 내보내기

- **HTML 내보내기**: **파일 → HTML 내보내기** 사용 — 대화형 VMark Reader 포함
- **PDF 내보내기**: 인쇄 (`Cmd/Ctrl + P`) 후 PDF로 저장
- **HTML로 복사**: `Cmd/Ctrl + Shift + C`

내보낸 HTML에는 목차, 설정 패널 등이 포함된 VMark Reader가 포함됩니다. [자세히 알아보기 →](/ko/guide/export)

## 설정

`Cmd/Ctrl + ,`로 설정을 열어 다음을 사용자 지정합니다:

- **모양**: 테마, 폰트, 폰트 크기, 줄 높이
- **에디터**: 자동 저장 간격, 기본 동작
- **파일 및 이미지**: 에셋 관리, 문서 도구
- **통합**: AI 제공자, MCP 서버
- **언어**: CJK 서식 규칙
- **마크다운**: 내보내기 옵션, 서식 기본 설정
- **단축키**: 키보드 단축키 사용자 지정
- **터미널**: 터미널 폰트 크기 및 줄 높이

## AI 글쓰기 지원

VMark에는 AI 지니가 내장되어 있습니다 — 텍스트를 선택하고 `Mod + Y`를 눌러 AI로 글을 다듬고, 확장하고, 번역하거나 변환할 수 있습니다. **설정 > 통합** 에서 원하는 제공자를 구성하세요.

[AI 지니에 대해 자세히 알아보기 →](/ko/guide/ai-genies) | [제공자 구성 →](/ko/guide/ai-providers)

## 시작을 위한 팁

1. **아웃라인으로 탐색**: 아웃라인 항목을 클릭하여 섹션으로 이동
2. **포커스 모드 사용**: `F8`은 현재 단락을 제외한 모든 것을 어둡게 만듭니다
3. **글을 쓰면서 검증**: `Cmd + Shift + L`은 마크다운 린트 엔진과 깨진 링크 검사를 실행합니다
4. **단축키 학습**: 전체 참조는 [단축키 가이드](/ko/guide/shortcuts)에 있습니다
